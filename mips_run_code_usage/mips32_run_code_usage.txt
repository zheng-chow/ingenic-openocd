1、指令格式
monitor mips32 run_code [file_name] elf
例如您需要执行的可执行程序为test.exe，则命令如下：
monitor mips32 run_code test.exe elf 注意目前仅支持elf格式
该命令会在openocd所处系统的根目录寻找该可执行程序

2、可执行程序的入口地址必须为0xff200200；

3、可执行程序不能存在访问dmseg区域的load/store指令；

4、目前可执行程序的大小为0xff200200~0xff202000,之后会进一步扩展，如果可执行程序的代码段不在此区间内将导致未知错误；

4、可执行程序中只能存在一个段（section）；

5、在可执行程序的末尾一定要再次跳转到0xff200200,openocd以此作为结束的标志；

6、可参照当前目录下的链接脚本以及测试程序来组织您的可执行程序；
